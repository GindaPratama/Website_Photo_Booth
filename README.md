# Website Photo Booth 📸

Selamat datang di repositori proyek **Website Photo Booth**!

Ini adalah aplikasi web photo booth modern yang sedang dalam tahap pengembangan aktif. Tujuan utama dari proyek ini adalah untuk menciptakan pengalaman photo booth yang **kaya fitur**, menyenangkan, interaktif, dan yang terpenting, **100% gratis** untuk digunakan oleh siapa saja, langsung dari browser mereka.

## 🌟 Tentang Proyek Ini

Proyek ini berawal dari keinginan untuk menyediakan alternatif alat photo booth yang mudah diakses tanpa biaya tersembunyi, langganan, atau watermark yang mengganggu. Kami ingin menghadirkan keseruan photo booth fisik ke dalam dunia digital dengan fitur-fitur kreatif yang lengkap.

Desain antarmuka pengguna (UI) dan pengalaman pengguna (UX) untuk proyek ini didasarkan pada mockup yang telah dirancang dengan cermat di **Figma**.

## 🚀 Fitur Utama (Direncanakan)

Berikut adalah beberapa fitur unggulan yang sedang dan akan kami kembangkan:

- **📸 Pengambilan Gambar Langsung:** Menggunakan webcam atau kamera perangkat untuk mengambil foto berkualitas tinggi.
- **⏱️ Timer & Mode Burst:** Hitung mundur yang dapat disesuaikan dan mode pengambilan gambar beruntun (misal: 3 atau 4 pose sekaligus).
- **✨ Filter & Efek Real-time:** Beragam pilihan filter warna, efek vintage, dan efek visual lainnya yang dapat dilihat langsung sebelum mengambil foto.
- **🖼️ Bingkai & Kolase Kustom:** Pilihan tata letak (grid) foto (strip 2x6, grid 4x4, dll.) dan bingkai tematik yang menarik.
- **🎨 Alat Pengeditan Cepat:** Fitur untuk menambahkan stiker lucu, teks, atau coretan tangan setelah foto diambil.
- **💾 Simpan & Bagikan Mudah:** Unduh hasil foto ke perangkat lokal dalam berbagai format atau bagikan langsung ke platform media sosial.
- **🆓 Sepenuhnya Gratis:** Komitmen kami adalah menjaga alat ini 100% gratis selamanya. Tidak ada fitur premium yang dikunci.

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun di atas fondasi teknologi web modern untuk menjamin performa yang cepat dan pengalaman pengembang yang baik:

- [**React**](https://react.dev/): Library JavaScript untuk membangun antarmuka pengguna.
- [**Vite**](https://vitejs.dev/): Build tool front-end yang sangat cepat, digunakan untuk inisialisasi dan pengembangan proyek.
- **CSS Modern / Styling Library** (Akan ditentukan nanti, misal: Tailwind CSS, Styled Components, atau CSS Modules) untuk mengimplementasikan desain dari Figma.

## 💻 Cara Menjalankan Secara Lokal (Getting Started)

Jika Anda ingin menjalankan proyek ini di komputer Anda untuk tujuan pengembangan atau pengujian, ikuti langkah-langkah berikut:

**Prasyarat:** Pastikan Anda telah menginstal [Node.js](https://nodejs.org/) di komputer Anda.

1.  **Clone repositori ini** (jika sudah di-host di GitHub/GitLab):

    ```bash
    git clone [URL_REPOSITORI_ANDA]
    cd website-photo-booth
    ```

    _Jika belum ada repositori remote, cukup buka folder proyek Anda di terminal._

2.  **Instal dependensi:**

    ```bash
    npm install
    ```

3.  **Jalankan server pengembangan:**

    ```bash
    npm run dev
    ```

4.  Buka browser Anda dan akses URL yang muncul di terminal (biasanya `http://localhost:5173`). Anda akan melihat halaman awal aplikasi (saat ini mungkin masih halaman default Vite + React sampai Anda mulai mengeditnya).
